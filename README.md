# 图书管理系统 java762

## 项目概述

图书管理系统是一个由大学生独立开发的应用，致力于提供独特的图书管理解决方案。

## 技术栈

### 后端

- **Python**: 负责业务逻辑处理、数据操作、与前端的交互。
- **Flask**: 轻量级Web框架，用于搭建网站后端、设计API路由、处理用户请求。

### 数据库

- **MySQL**: 关联型数据库，用于存储用户、图书、借阅、公告等业务数据。

### 前端

- **Bootstrap 5**: 前端UI框架，负责页面布局和样式设计。
- **HTML/CSS/JavaScript**: 前端基础技术，负责页面结构、样式和交互效果。
- **FontAwesome**: 图标库，提供系统各模块的现代化图标。

## 功能模块

- 用户登录
- 图书管理
- 借阅管理
- 读者管理
- 公告管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.x
- Flask 2.x
- MySQL 8.x
- Bootstrap 5
- 兼容主流现代浏览器
